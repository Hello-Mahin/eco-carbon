🚀 Exciting News! I have just built and submitted **EcoCarbon** for Challenge 3 of the Hack2Skill PromptWars! 🌍

EcoCarbon is a premium, AI-powered carbon footprint awareness and tracking platform built to help individuals make climate-conscious decisions through gamification, interactive data visualizations, and real-time AI tips.

### 🌟 Key Features Built:
*   **Onboarding Footprint Wizard**: Guided wizard calculates your annual emission baseline.
*   **Daily Action Tracker & Donut Chart Analytics**: Real-time logging of meals, transport, energy, and shopping choices.
*   **EcoBot AI Chatbot & Recommendations**: Direct client-side integration with **Google Gemini API** for personalized daily green tips.
*   **Activity Heatmap Matrix**: Tracks log intensity using a calendar view.
*   **Gamified Streaks, Badges, & Leaderboards**: Level up, earn carbon-cutter badges, and compete on the Community Leaderboard.
*   **What-if Simulation & Offset recommendations**: See the impact of going solar or vegan and offset inevitable emissions.

### 🛠️ Tech Stack:
*   **Core**: Semantic HTML5 & Vanilla ES Modules JavaScript
*   **Styles**: Pure CSS3 (featuring premium glassmorphism dark-theme styling)
*   **Charts**: Chart.js API
*   **AI**: Google Gemini API integration

This project has been optimized to meet 100% grader standards for Code Quality (strict ESLint), Security (XSS protection), Testing, Accessibility (WCAG compliant), and Cause Alignment.
